# Blinkit Sales Data Analysis (Power BI)

## Project Overview
This project analyzes Blinkit sales data using Power BI to identify revenue trends, product performance, and customer purchasing behavior.

## Tools Used
- Power BI
- DAX
- Data Cleaning
- Data Modeling

## Key Analysis
- Sales performance by category
- Top performing products
- Revenue trends over time
- Profit analysis

## Dashboard Features
- KPI cards for revenue and profit
- Category-wise sales breakdown
- Time-based trend analysis
- Interactive filters and slicers

## Key Insights
- Identified high-performing product categories
- Seasonal sales trends
- Profitability differences between product segments

## Files
- `Blinkit_Dashboard.pbix` – Power BI dashboard
- `dataset.csv` – raw dataset
- `insights_report.pdf` – detailed analysis
